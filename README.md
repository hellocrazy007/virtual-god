# 五行


#### 源码介绍
功能包括：命理推算，八字精批，每天运气，时时运气，塔罗占卜，星座运势，虚拟大神（机器人，后面单独发布，可以做大仙主播，虚拟人聊天测算提醒等）

#### 源码结构

- wuxing-admin 后台源码
- wuxing-api 接口源码
- wuxing-open-api 接口开放平台
- wuxing-uniapp 移动端源码
- wuxing-vue-admin 后台前端源码


#### 环境搭建
以下可以用宝塔面板安装，如果没有宝塔，请自行编译安装或yum安装

- jdk 1.8+
- mysql 8.0+
- nginx最新
